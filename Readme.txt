MyResume Portfolio Website
--------------------------

Description:
This is a personal resume and portfolio website built with HTML, CSS, and JavaScript. It showcases my professional background, design projects, and services.

Folder Structure:
- index.html                  --> Home page
- portfolio-details.html      --> Portfolio project details
- service-details.html        --> Service-related content
- starter-page.html           --> Template starting point
- assets/
   ├── css/                   --> Main styling files
   ├── img/                   --> All images and icons
   ├── js/                    --> JavaScript functionality
   └── vendor/                --> Third-party libraries (e.g., AOS)

Instructions for Deployment:
1. Upload all files (including index.html) to the root of your GitHub repository.
2. Go to GitHub > Settings > Pages.
3. Select the branch: `main` and folder: `/ (root)`
4. Click Save and wait a minute.
5. Your site will be live at: https://your-username.github.io/repo-name/
